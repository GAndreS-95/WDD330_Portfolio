Variables

Variables store value in memory for specific uses in JavaScript. Use keywords like var, let, and const to use variables. Naming variables can start with any upper/lower case letters, underscores, or dollar symbols. It can contain numbers but can't start w/ them. Variables are also very case sensitive.

Reserved Words

They are words in JavaScript that are reserved and thus can't be named with such words.

Assignment

Assigning values to a variable is done by using the [ = ] operator.
We can also declare and initialize a variable at the same time.
i.e. let name = "Howard";

We use the dot [ . ] operator followed by the name of the method, followed by parentheses to call methods.

i.e.
name.toUpperCase(); --- Changes object to all upper case letters.
name.toLowerCase(); --- Changes object to all lower case letters.

Bracket Notation can also be used to access a primitive value's properties using the square brackets '[ ]'

i.e. name.length;
>>> 10

Number Methods

Numbers also have some built-in methods, although you need to be careful when using the dot notation with number literals that are integers because the dot can be confused for a decimal point. [either use two dots, add a space, write the integer as a decimal, or place the integer in parentheses]

'toFixed( )' - this method rounds a number to a fixed number of decimal places.

'toPrecision( )' - this method rounds a number to a fixed number of significant figures that is returned as a string.

NaN

NaN is an error value that is short for "Not a Number". It is used when an operation is attempted and the result is not numerical.

Converting strings to numbers & vice versa

Strings to Numbers

  • Coercion -- multiply a numerical string by 1.
  • Place a [+] symbol in front of the numerical string.
  • Number function -- This is the preferred way to convert strings into numbers as it avoids type coercion in the background. The conversion is explicit, making it obvious what is being done.
    ie. Number("23");

Numbers to Strings

  • Adding numbers to empty stings
    i.e. 3 + ' ';
  • String function -- most preferred method.
    i.e. String(3);
  • toString( ) method.

Parsing Numbers

The function parseInt( ) can be used to convert a numerical string into an integer.

Array Properties and Methods

Arrays are a powerful weapon in a JavaScript ninja s toolkit and have some useful methods.

Pop, Push, Shift, and Unshift

To remove the last item from an array, we can use the pop() method.
The push() method appends a new value to the end of the array.
The unshift() method is similar to the push() method, but this appends a new item to the beginning of the array.

The join() Method

The join() method can be used to turn the array into a string that comprises all the items in the array, separated by commas.
You can choose a separator other than a comma by placing it inside the parentheses.

Slicing and Splicing

The slice() method creates a subarray, effectively chopping out a slice of an original array starting at one index and finishing at the next.

Note that this operation is non-destructive -- no items are actually removed from the array.
The splice() method removes items from an array and then inserts new items in their place.
The splice() method can also be used to insert values into an array at a specific index without removing any items, by indicating that zero items are to be removed
If you want to remove a value completely, you can use the splice() method with a length of 1 and without specifying any values to add.

Reverse

We can reverse the order of an array using the reverse() method.
Note that this changes the order of the array permanently.

Callbacks

Remember at the start of this chapter when we said that functions in JavaScript are first-class objects, and this means that they behave in just the same way as every other value? Well, they can also be given as a parameter to another function. A function that is passed as an argument to another is known as a callback.
